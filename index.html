Model Context Protocol

Use Vik from Claude, ChatGPT, Cursor.

Big Vik Claw ships as an MCP server. One config line in your AI client, Vik's tools show up.

Claude Desktop config (claude_desktop_config.json)
{
  "mcpServers": {
    "vik": {
      "command": "npx",
      "args": ["-y", "@fizznetwork/vik-mcp-server"]
    }
  }
}

Restart Claude Desktop. Open any chat. Ask "list my scheduled posts" — Claude calls Vik's tools.
